srvPrettify

This directive grabs the html contents of the tag and prettifies them. If the tag contains the info about the file name, it gets the text from the file, prettifies and places inside the container. It uses bootstrap's prettify().

Examples: All the code snippets of the pages are prettified using this directive.
  • Usage
  • srvPrettify.js
<srv-prettify>Your code</srv-prettify> <srv-prettify file-name="your file name"></srv-prettify>
